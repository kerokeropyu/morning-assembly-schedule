【朝礼当番メンバーの変更方法（事務ご担当者さま向け）】

この説明は、朝礼当番表に出てくるメンバーを変更するための手順です。
むずかしい専門知識は不要です。

--------------------------------------------------
1. 設定ファイルを見つける
--------------------------------------------------
1) 朝礼当番表（Excel）を自動で作っているフォルダを開きます。
   （中に「generate_schedule.py」というファイルがある場所です）
2) その中に「morning_assembly_config.json」というファイルがあります。
   これが「設定ファイル」です。

--------------------------------------------------
2. 設定ファイルを開く（Windows・メモ帳）
--------------------------------------------------
1) 「morning_assembly_config.json」の上で右クリックします。
2) 「プログラムから開く」→「メモ帳」を選びます。
3) 文字がたくさん並んだ画面（メモ帳）が開きます。

--------------------------------------------------
3. どこを変更すればよいか
--------------------------------------------------
開くと、次のような内容が見えます。

{
  "department_name": "総務部",
  "members": [
    "田中 太郎",
    "佐藤 花子",
    "鈴木 一郎",
    "高橋 美咲",
    "伊藤 健太"
  ],
  "output_directory": "./morning_assembly_schedules",
  "excel_filename": "総務部朝礼当番表.xlsx"
}

この中で、事務の方が変更してよいのは主に以下の2か所です。

(1) "members": [...] の部分 → 朝礼当番に出したい人の名前
(2) "department_name": "総務部" → 部署名（必要な場合のみ）

それ以外の行は、基本的に変更しないでください。

--------------------------------------------------
4. メンバーを追加する（人を増やす）
--------------------------------------------------
例：「山田 次郎」さんを追加したい場合

変更前：
"members": [
  "田中 太郎",
  "佐藤 花子",
  "鈴木 一郎",
  "高橋 美咲",
  "伊藤 健太"
],

変更後：
"members": [
  "田中 太郎",
  "佐藤 花子",
  "鈴木 一郎",
  "高橋 美咲",
  "伊藤 健太",
  "山田 次郎"
],

ポイント：
・名前は "山田 次郎" のように、両側を半角の「"」で囲みます。
・一番下の人の行には「,（カンマ）」を付けないようにします。
　（上の人の行にはカンマが付いていてOKです）

--------------------------------------------------
5. メンバーを削除する（人を減らす）
--------------------------------------------------
例：「鈴木 一郎」さんを当番から外したい場合

変更前：
"members": [
  "田中 太郎",
  "佐藤 花子",
  "鈴木 一郎",
  "高橋 美咲",
  "伊藤 健太"
],

変更後：
"members": [
  "田中 太郎",
  "佐藤 花子",
  "高橋 美咲",
  "伊藤 健太"
],

「鈴木 一郎」の行を丸ごと削除してください。

削除したあと、一番下の人の行だけカンマ無しになっていることを確認してください。

--------------------------------------------------
6. 名前を修正する（結婚・表記変更など）
--------------------------------------------------
例：「佐藤 花子」さんが「山本 花子」さんになった場合

変更前：
"佐藤 花子",

変更後：
"山本 花子",

ダブルクォーテーション（"）の中だけ書き換えてください。
前後のカンマやスペースはそのまま残してください。

--------------------------------------------------
7. 部署名を変更する（必要な場合のみ）
--------------------------------------------------
"department_name": "総務部",

と書かれている部分があります。

部署名を「管理部」に変更したい場合は、

"department_name": "管理部",

のように、"総務部" の中だけを書き換えてください。

--------------------------------------------------
8. 変更を保存する
--------------------------------------------------
1) メモ帳の上のメニューから「ファイル」→「上書き保存」をクリックします。
   （またはキーボードで「Ctrl」キーを押しながら「S」キー）
2) 右上の「×」ボタンでメモ帳を閉じます。

これで設定の変更は完了です。
次にバッチ（run_monthly_batch.bat など）を実行したときから、
新しいメンバー・部署名で朝礼当番表が作られます。

--------------------------------------------------
9. うまくいかない／不安なとき
--------------------------------------------------
・編集中に「やっぱりやめたい」と思った場合は、
  メモ帳を閉じるときに「保存しない」を選べば、元に戻せます。

・どうしても不安な場合は、
  変更前と変更後の画面をスクリーンショットして、
  システム担当者に確認してもらってください。
